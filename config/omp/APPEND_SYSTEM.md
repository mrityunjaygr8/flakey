# Browser tool selection
When the user asks to open a URL interactively (e.g., "open this site", "show me", "browse"), use the Chrome DevTools MCP browser, not the built-in `browser` tool. The MCP browser is for interactive user-facing browsing where visual rendering matters.

When browsing for research, automation, scraping, or reading page content programmatically, use the built-in `browser` tool instead.
